### TicketMonster

TicketMonster is a side project inspired by TicketMaster, designed to simulate a complete ticket booking system. The project utilizes ASP.NET Core MVC for backend development and employs modern frontend techniques like Ajax, Axios, and jQuery. The system features a layered architecture with Dependency Injection, ensuring a clean and maintainable codebase.
**Key Features:**

- **Frontend:**

  - **Venue Design:** Built with Adobe Illustrator and SVG, enabling dynamic drag-and-drop, zoom, and click functionalities for an enhanced user experience.
  - **Seat Selection:** Includes row and seat filtering with data-driven UI updates using ViewModel.
  - **Event Search:** Implemented a search box with Razor Form and data handling through Repository and Service patterns.

- **Backend:**
  - **Data Management:** Leveraged LINQ and WebAPI for data filtering and efficient data handling with Entity Framework Core and Dapper.
  - **Admin Analytics:** Provided administrators with insightful charts using Vue.js and ECharts, powered by Axios and WebAPI for data requests.
